Das {PLUGINNAME} Plugin erzeugt eine Karte mit Infos
sowie einen Link zu einem Routingservice mit vorbelegter Zieladresse.

Beispiele:

Karte mit Point

{{PLUGINNAME}| lat=(Latitude), long=(Longitude) }
Optional: width, height, zoom, layers, satellite, full, scale.

Karte mit Point und Text

{{PLUGINNAME}| lat=(Latitude), long=(Longitude), infotext=(Text) }
Optional: width, height, zoom, layers, satellite, full, scale.

Karte mit eigenem Icon und Text

{{PLUGINNAME}| lat=(Latitude), long=(Longitude), infotext=(Text), icon=(Datei oder URL)}
Optional: width, height, zoom, layers, satellite, full, scale, iconsize.

Karte mittels KML oder GPX Datei

{{PLUGINNAME}| KML oder GPX Datei }
Optional: width, height, layers, satellite, full, scale.

Google Routingservice-Link

{{PLUGINNAME}| google=(Text), linktext=(Text) }

OpenRouteService Routingservice-Link

{{PLUGINNAME}| openrouteservice, linktext=(Text), lat=(Latitude), long=(Longitude) }
Optional: zoom.

OpenStreetMap Routingservice-Link

{{PLUGINNAME}| openstreetmap, linktext=(Text), lat=(Latitude), long=(Longitude) }
Optional: zoom.

Parameter:

  • lat (Latitude) geographische Breite (früher Breitengrad)
  • long (Longitude) geographische Länge (auch Längengrad oder nur Länge)
  • height 350px (Zahl) oder (Zahl CSS-konform)
  • width (Zahl) oder (Zahl CSS-konform)
  • zoom 10 (Zahl) Voreinstellung Zoomlevel
  • infotext (Text) mozilo-Syntax ist erlaubt. Achtung: , oder = oder " müssen geschützt werden
  • icon mozilo-Dateisyntax oder URL mit http(s):// am Anfang
  • iconsize (Zahl) Höhe und Breite oder (Zahl)x(Zahl) Höhe x Breite das x ist das Trennzeichen
  • linktext (Text) mozilo-Syntax ist erlaubt. Achtung: , oder = oder " müssen geschützt werden
  • google (Latitude Longitude oder eine Adresse) Achtung: , und = müssen geschützt werden
  • openrouteservice (ohne Parameter)
  • openstreetmap (ohne Parameter)
  • openinfo (ohne Parameter) infotext öffnen
  • satellite (ohne Parameter) Satellitenansicht
  • layers (ohne Parameter) Kartenauswahl
  • full (ohne Parameter) Fullscreen Button aktiv
  • scale (ohne Parameter) Anzeige Maßstab aktiv

Zeichen schützen / maskieren in moziloCMS erfolgt über ein vorangestelltes Zirkumflex ugs. Hochdach.

Tipp: Übersichtlichere Plugin-Parameter

Die Parameter können jeweils in einer eigenen Zeile stehen, auch Leerzeichen sind erlaubt.

Tipp: Eigene KML oder GPX Datei erstellen

Via My Maps von google kann eine eigene KML Datei erstellt werden.
Einfach "In KML exportieren" anklicken und speichern, danach muss die Datei noch entpackt werden.
Alternativ mit einem freien Editor uMap der kann KML und GPX.

Tipp: Ermitteln von Latitude und Longitude

Ort auf OpenStreetMap eingeben, auf Teilen gehen und "Kartenmarker setzen" anklicken, die Latitude und Longitude steht nun in dem "Link" Feld
